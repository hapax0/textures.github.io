Brush/Texture tool

Use this tool to create brushes and patterns for use with Concepts app or similar.

Brushes (top row of icons: left to right)

The brushes and patterns mostly use the "feature size", "quantity" and "opacity" sliders.
  • Airbrush: use "quantity" and "feature size" to adjust output. This is the only brush that uses the guassian distribution.
  • Haze: fills the canvas with a random lnear gradient.
  • Rips: simulates ripped paper. Uses "feature size" and opacity.
  • Mass: irregular shape on left or right.
  • Blobs: small irregular shapes.
  • Smudges, Fiber, Horizontal and Arcs: also use "quantity" and "feature size".
  • Halftone and the four hatches brushes use "closeness" settings to pscae out the lines or dots.
  • Skinny lines: horizontal lines across the whole canvas.
  • Rays: lines radiating from a small area in the canvas.
  • Spirals: two types of spiral lines.
  • Clear the canvas.
  • Download the canvas as a PNG file.

Other functions

Load image... select an image file to load in the canvas.

Background: color is not saved with the PNG file. Toggle to black or gray when working with light colors.

Chars: enter a single character, click chars and the canvas is filled with the character.

Text tool: links to the Text tool page, where you can generate text in different styles and colors.

Test: select a function from the list, then click to apply it.

Grungy patterns: when this is enabled most effects are modified - generally adding more randomness to the result.

Mode: refer to globaCompositeOperation and use the default unless you know how to use this feature. Use destination-over to draw only on transparent areas. Usefull if you want to fill in a background withot overwriting existing drawing.

Distribution

  • Circle. Useful for creating a brush shape.
  • Square: for a background texture. Smudges, halftones and hatches should produce seamless patterns with most settings. Enabling Grungy patterns can break seamlessness.
  • Gaussian: only applies to the airbrush tool.
  • Flatten: clips the canvas to flatter region.

How to make a paper-like texture

  • Clear the canvas and set the background color to gray.
  • Set feature size, quantity and opacity sliders to the middle.
  • Set the paint color to white.
  • Click the "fiber" button two or three times.
  • Change the paint color to black.
  • Click the "fiber" button two or three times.
  • Experiment with the three slider settings and try the different fiber buttons.
